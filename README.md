**Penjelasan:**

- `username`: username target contoh `ant10`
- `message`: pesan yang dikirim.
- `count`: jumlah pesan yang dikirimkan.
- `user-agent`: user-agent.
- `threads`: banyak eksekusi.
- `use_proxies`: penggunaan proxy (ya/tidak)
